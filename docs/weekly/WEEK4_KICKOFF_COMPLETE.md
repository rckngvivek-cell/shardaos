## 🚨 MOST COMMON WEEK 4 MISTAKES

**Don't make these:**

1. ❌ **Skipping plan phase**
   - ✅ Instead: Write plan before coding

2. ❌ **Merging without tests**
   - ✅ Instead: Tests first, then merge

3. ❌ **Forgetting auth middleware**
   - ✅ Instead: All endpoints require bearer token

4. ❌ **No error handling**
   - ✅ Instead: Return proper HTTP status codes (400/401/404/500)

5. ❌ **Hard-coded credentials**
   - ✅ Instead: Use `.env` files, secrets manager

6. ❌ **Skipping code review**
   - ✅ Instead: All PRs reviewed before merge

7. ❌ **No monitoring**
   - ✅ Instead: Cloud Logging live by Thursday
