## 🎯 SALES CALL STRUCTURE

### Call Duration: 15 minutes

**Agenda:**
```
MINUTE 1:  Warm-up & intro
MINUTE 2:  Context about pilot program
MINUTES 3-8: Live product demo
  - Show: Student enrollment
  - Show: Attendance tracking  
  - Show: Grade entry
  - Show: Admin dashboard
MINUTES 9-13: Q&A on pricing, training, timeline
MINUTE 14: Next steps (pilot agreement)
MINUTE 15: Closing
```

### Demo Script
```
"Let me quickly show you the system. [Share screen]

First, here's how a school admin enrolls students. 
[Demo student enrollment form]

Once students are in the system, teachers can mark attendance 
in real-time. [Show attendance dashboard]

And here's how grades are entered and reported to students. 
[Show grade entry interface]

Finally, this is the principal dashboard - all the school's 
metrics at a glance. [Show dashboard]

Questions? What do you think would be most valuable for your school?"
```

### Key Talking Points

**Feature Highlights:**
- ✅ **Real-time data**: All changes visible instantly
- ✅ **Mobile-friendly**: Works on smartphones, tablets, laptops
- ✅ **Secure**: Bank-level encryption, role-based access control
- ✅ **Easy setup**: 30-minute admin training, we handle migration
- ✅ **Support**: Direct support team during beta

**Addressing Objections:**
- *"How much will it cost?"* → "For now, completely free during beta. Post-beta, we're exploring affordable pricing for schools like yours."
- *"Can I import existing data?"* → "Yes! Send us a sample of your student data, we'll do the migration for you."
- *"What if we don't like it?"* → "Totally fair. We'll have your data export ready at any time. Zero long-term commitment."
- *"How do you handle data security?"* → "We're SOC 2 compliant, encrypted at rest, daily backups, regular security audits."

**Next Steps Close:**
"Great! I'd like to get [School Name] live on this system by April 13. 
Can I send you a pilot agreement to review? Once you sign, we'll:
1. Set up your admin account
2. Load your student data
3. Train your team on April 12
4. Go live!"

## 💬 FEEDBACK COLLECTION TEMPLATE

### WEEKLY FEEDBACK CALL (30 min)

**Questions to Ask:**

1. **Overall Experience (5 min)**
   - "On a scale 1-10, how easy was it to use the system?"
   - "What worked well?"
   - "What was frustrating?"

2. **Feature Usage (5 min)**
   - "Which features did you use most?"
   - "Which features did you NOT use?"
   - "Why did you skip certain features?"

3. **Issue Logging (5 min)**
   - "Did you encounter any bugs?"
   - "How did you report them?"
   - "Were they resolved quickly?"

4. **Feature Requests (10 min)**
   - "What features would make this more valuable?"
   - "What would improve teacher experience?"
   - "What would improve parent/student experience?"

5. **Competitive Context (3 min)**
   - "Have you used other school systems?"
   - "How does this compare?"
   - "What can we learn from them?"

6. **Likelihood to Recommend (2 min)**
   - "Would you recommend this to other schools?"
   - "What would make it a definite yes?"
