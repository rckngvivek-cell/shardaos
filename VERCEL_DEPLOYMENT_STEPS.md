# 🚀 Deploy to Vercel - Step by Step

## FASTEST WAY (2 minutes)

### 1. Go to Vercel Website
Open in browser: https://vercel.com

### 2. Click "New Project"
Connect your GitHub repository and Vercel will auto-deploy!

---

## OR USE CLI (3 minutes)

### 1. Open Terminal
```powershell
cd c:\Users\vivek\OneDrive\Scans\files
```

### 2. Login to Vercel
```powershell
vercel login
```
- Opens browser automatically
- Authorize and return to terminal
- ✅ You're logged in!

### 3. Deploy to Production
```powershell
vercel --prod
```

**Result**: 🎉 Your app is LIVE!
- URL: `https://school-erp.vercel.app`
- Dashboard: `https://vercel.com/dashboard`
