This document defines the roles, responsibilities, routines, and workflows for all specialized co-working agents. Each agent owns specific technical specifications and must enforce PRI Framework on all PRs.

This organizational structure is built for **multi-billion dollar company excellence**, with:
- 1 Senior Team Leader (Architecture, Strategy, Team Management)
- 6 Specialized Expert Agents (each at god-level expertise)
- All agents follow PRI Framework (Plan → Review → Implement)
- Rigorous performance management, quality assurance, and continuous improvement

## Communication Channels & Frequency

### **Daily Standup (Async)**
- **Channel:** Shared workspace (Slack/Discord)
- **Frequency:** Daily, agent posts when starting work
- **Content:** What I'm doing, what I need, blockers
- **Format:** 3-5 sentences, no long discussions
- **Archived:** Auto-saved with daily snapshot

### **Blocker Resolution (Real-time)**
- **Channel:** Dedicated #blockers channel
- **Response Time:** Within 1 hour
- **Owner:** Agent affected + dependent agents
- **Escalation:** If not resolved in 2 hours, escalate

### **Code Review (Async)**
- **Channel:** GitHub/GitLab PR comments
- **Response Time:** Within 4 business hours
- **Comments Archived:** Auto-saved with PR

### **Phase 2: Work Completion (Individual Agent)**
- **Duration:** 2-3 hours per agent
- **Output:** Summary document + archived chats
- **Saved Location:** /agent-archives/{agent}/week-{N}/

### **Phase 3: Project Sync (Team Meeting)**
- **Frequency:** Once per week, after Phase 2 done
- **Duration:** 1-2 hours
- **Participants:** All agents + stakeholders
- **Format:** Structured presentations + discussion
- **Recording:** Video recorded + transcript saved
- **Minutes:** Notes saved to /project-docs/week-{N}/

### **Phase 4: Sprint Planning (Team Meeting)**
- **Frequency:** Once per week, before next week starts
- **Duration:** 1-2 hours
- **Participants:** All agents + stakeholders
- **Format:** Structured planning + commitment confirmation
- **Recording:** Video recorded + transcript saved
- **Output:** Sprint plan document saved

## Summary: Complete Weekly Cycle for Agents

**Every agent follows this 4-phase routine:**

1. **Phase 1:** Work on assigned tasks (Variable duration) → Parallel work with real-time collaboration
2. **Phase 2:** Capture learnings & archive work (2-3 hours) → After completing all tasks
3. **Phase 3:** Sync with team & discuss (1-2 hours) → Team meeting after all agents complete Phase 2
4. **Phase 4:** Plan next week (1-2 hours) → Team meeting before starting next week

**Then repeat for next week.**

This ensures:
- ✅ Continuous progress on tasks
- ✅ Knowledge captured from each week's work
- ✅ Team alignment on blockers & dependencies
- ✅ Clear plans for next iteration
- ✅ Continuous improvement through metrics & learnings
- ✅ Strong cross-agent collaboration
- ✅ Full audit trail of decisions & work
