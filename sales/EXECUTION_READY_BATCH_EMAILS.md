# READY-TO-SEND EMAIL BATCH - Copy & Paste into Gmail/Outlook

**Instructions:**
1. Open Gmail or Outlook
2. Click "Compose" → "New Message"
3. Add TO field: Select all 10 recipients at once (or send individually for more personalization)
4. Select each recipient's personalized email below
5. CTRL+C to copy entire email
6. CTRL+V into compose window
7. Click SEND

## ⚡ SENDING OPTIONS

### Option A: Gmail Mail Merge (Fastest)
1. Open Gmail
2. Click "Compose"
3. Use "To" field to add all 10 emails
4. Copy/paste first personalized email
5. Send → Repeat for each school
**Time:** 30 minutes

### Option B: Outlook Mail Merge
1. Create contacts for all 10 principals
2. Use Tools → Mail Merge → select this file
3. Merge to email
**Time:** 20 minutes

### Option C: Manual Send (Most Personal)
Send individually to each principal for higher engagement
**Time:** 45 minutes but highest response rate

**Recommended:** Option A + send individually = 45 min, highest ROI
