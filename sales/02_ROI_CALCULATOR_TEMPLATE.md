## ROI CALCULATOR SPREADSHEET (roi-calculator.csv)

```
INPUT FIELDS,,,
School Name,Delhi Public School,
Student Count,3000,
Current Admin Costs (hours/month),150,
Current Parent Satisfaction (1-10),5,
Current Engagement (%),45,
,,,
CALCULATIONS & OUTPUT,,,
,,,
Annual Admin Time Savings,,"=([Current Admin Costs] * 12) * 100"
Admin cost per hour,,"₹300"
Annual rupees saved from admin time,,"=[Annual Admin Time Savings] * 300"
,,,
Satisfaction Improvement (points),,"3"
Engagement Improvement (%),,"25%"
Engagement lift in actual parent interactions,,"=[Student Count] * [Current Engagement] * [Engagement Improvement]"
,,,
FINANCIAL SUMMARY,,,
,,,
Annual Software Cost,,"₹2,80,000"
Annual Admin Time Savings,,"=See above calculation"
Annual Engagement/Satisfaction Benefit,,"₹50,000 (estimated from retention)"
,,,
Total Annual Benefits,,"=Admin Savings + Engagement Benefit"
Total Annual Cost,,"₹2,80,000"
,,,
NET ANNUAL SAVINGS,,"=Total Benefits - Cost"
Payback Period (weeks),,"=6"
,,,
3-YEAR PROJECTION,,,
Year 1 Investment,,"₹2,80,000"
Year 1 Benefit,,"=NET ANNUAL SAVINGS * 1"
Year 2-3 Benefit (per year),,"=(NET ANNUAL SAVINGS + 20%) * 2"
,,,
3-Year Net Benefit,,"=Cumulative benefits across 3 years"
3-Year ROI,,"=((3-Year Benefit - 3-Year Cost) / 3-Year Cost) * 100"
```
